## Como Usar

```bash
# Clone o repositório
$ https://github.com/MuriloCicolin/Recipe-App

# Vá para pasta do repositório
$ cd Recipe-App

# Instale as dependências 
$ yarn install

# Rode a Aplicação
$ yarn start
```
